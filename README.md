## WEB SCRAPER FOR PROKERALA CALENDAR DATA

- rename `.env.local.template` to `.env.local` and set the `BRIGHT_DATA` value to `Bright data browser endpoint` [visit beight data](https://brightdata.com/cp/zones/scraping_browser1/stats?id=hl_470e2004)

```bash
mv .env.local.template .env.local
```

- run the script

```bash
node index.js
```
